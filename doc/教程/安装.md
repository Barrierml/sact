# 兼容性
sact 仅支持能够使用ES 6的浏览器内使用
## 浏览器支持列表
1. Chrome：51 以上
2. Firefox：53 以上
3. Safari：10 以上
4. IE：Edge 15以上

## 使用`<script>`引入
先下载dist中的 **[Sact.js](../../dist/Sact.js)** 最新版本并用 `<script>` 标签引入，Sact 会被注册为一个全局变量。
```html
<script src="./Sact.js"></script>
```

## NPM

如果想在npm环境下使用sact可以先将项目克隆至本地目录下进行引入使用。
